Task manager written in Laravel

Deployed project: [Task Manager](https://task-manager-laravel-ln.herokuapp.com/)

## Requirements

* Make
* Docker and Docker Compose

## Setup

```bash
make compose-setup
```

## Start app locally

```bash
make compose
```

## Run tests and linter

```bash
make compose-test
```
